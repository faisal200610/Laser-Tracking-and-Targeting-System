Laser-Tracking-and-Targeting-System

Overview

This project is a real-time laser tracking and targeting system developed using Arduino Uno, Python, OpenCV, servo motors, and a smartphone IP camera.

The system detects and tracks moving objects through a live video stream. A pan-tilt mechanism controlled by servo motors continuously aligns a laser pointer with the detected target. A manually controlled launcher mechanism can then be activated by clicking the mouse within the tracking window.

This project demonstrates the integration of computer vision, object tracking, serial communication, servo motor control, and mechanical actuation in a single embedded system.

System Architecture

Camera System

A smartphone running the IP Webcam application provides a live video stream.

The IP address generated by the application is inserted into the Python code:

url = "http://YOUR_PHONE_IP:8080/video"

The camera should be mounted near the laser tracking system and aligned so that:

- The camera view is not inverted
- The laser point matches the center point of the video frame
- The camera and launcher share approximately the same direction

---

Tracking Process

1. Smartphone streams video using IP Webcam.
2. Python receives the video stream.
3. OpenCV detects moving objects.
4. The largest valid moving object is selected.
5. The target is locked.
6. Target position is smoothed.
7. Future movement is estimated.
8. Servo commands are generated.
9. Arduino receives commands through serial communication.
10. Pan and tilt servos align the laser with the target.

---

Launcher Mechanism

The launcher is constructed using:

- Wooden sticks
- Rubber bands
- Servo-based trigger

Operation

1. Load the projectile into the launcher.
2. Stretch and secure the rubber band.
3. The trigger servo holds the launcher in the loaded position.
4. The tracking system follows the target.
5. The laser indicates the current aiming position.
6. When the operator clicks the mouse on the tracking window:
   - Trigger servo rotates
   - Launcher releases
   - Projectile is launched
7. Reload manually for the next shot.

Wiring Summary

Arduino Pin Connections

Component| Arduino Pin
Pan Servo| D9
Tilt Servo| D10
Trigger Servo| D11
Laser Module| D7

Power

- Arduino powered through USB.
- Servos should preferably use an external regulated 5V supply.
- Common ground must be connected between Arduino and external power source.

How To Run

Step 1

Upload "Tracking.ino" using Arduino IDE.

Step 2

Connect Arduino to the computer.

Step 3

Install required Python libraries.

Step 4

Start IP Webcam on the smartphone.

Step 5

Copy the video URL.

Step 6

Paste the URL into:

url = "http://YOUR_PHONE_IP:8080/video"

Step 7

Run:

python Tracking.py

Step 8

Allow the system to detect and track a moving object.

Step 9

Click inside the tracking window to activate the launcher mechanism.
